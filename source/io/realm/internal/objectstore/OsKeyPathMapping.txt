io.realm.internal.objectstore
=============================


 | `java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 
 | 		io.realm.internal.objectstore.OsKeyPathMapping


**Implemented interfaces:**



- :ref:`io.realm.internal.NativeObject <io_realm_internal_NativeObject>` 


Wrapper for ObjectStore's KeyPathMapping class.The primary use case for this class is to make sure that we release native memory for any old mappings when the schema is updated.

It will use the already constructed ObjectStore schema to create the mappings required by the Query Parser.
